Sobre o simulador

O nosso simulador utiliza 3 tipos de bibliotecas:

PySimpleGUI é um pacote Python que permite que programadores Python de todos os níveis criem GUIs.

O NumPy é uma biblioteca para a linguagem Python com funções para se trabalhar com computação numérica.

A Python Matplotlib é uma biblioteca da linguagem de programação Python, utilizada para visualização de dados e plotagem gráfica.

Como utilizar?

Para utilizar o simulador é bastante simples: Com as informações recebidas através do roteiro, você só precisará preencher os campos de acordo. Não utilize virgulas ( , ) ou letras (ABC). Irá dar erro.
Após gerar a simulação, um gráfico será mostrado de acordo com as informações correspondentes.
